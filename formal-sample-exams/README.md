# 新增正式样题与参考答案

本文件夹放置两套按现有考纲重新组织的样题：

- `AI-Math-Formal-Sample-Exam.md`：人工智能数学基础正式样题
- `AI-Math-Formal-Sample-Answers.md`：人工智能数学基础参考答案
- `Pattern-Recognition-Formal-Sample-Exam.md`：模式识别与机器学习正式样题
- `Pattern-Recognition-Formal-Sample-Answers.md`：模式识别与机器学习参考答案

## 命题口径

这些题目按本仓库中的资格考试指南和复习笔记覆盖考点重新生成，不直接复制外部试题。题型参考了公开课程和高校课程页面中常见的期末题型分布，重点放在：

- 线性代数、概率论、最优化、前反向传播、自注意力等数学基础题；
- 贝叶斯决策、概率密度估计、KNN、EM、线性模型、聚类、PCA/LDA、SVM、决策树和综合应用等模式识别题。
